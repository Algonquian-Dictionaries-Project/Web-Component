Text Format Help

Here is how to format your plain text input text.

Each line should ideally contain one sentence, although that is not a strict rule.

Paragraph breaks are indicated by inserting a blank line.

Page breaks are indicated by inserting two consecutive blank lines.

Example:

This is sentence one in paragraph one on page one.
This is sentence two in paragraph one on page one.

This is sentence one in paragraph two on page one.
This is sentence two in paragraph two on page one.


This is sentence one in paragraph one on page two.
