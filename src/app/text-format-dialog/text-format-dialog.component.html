Text Format Help

Here is how to format your plain text input text.

Each line should ideally contain one sentence, although that is not a strict rule.

Paragraph breaks are indicated by inserting a blank line.

Page breaks are indicated by inserted two consecutive blank lines.

Example:

This is a sentence.
This is a second sentence.

This is a second paragraph.
Sent 2 in the second paragraph.


This text starts page 2.

A second paragraph on page 2.


Page 3 has this sentence.
And it has that sentence, in the same paragraph.
